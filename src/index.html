HEIC To PNG or JPG

Call one of these functions from FM:

convertHeicToPng (
  heicAsBase64: string,
  fmScript: string,
  scriptOption?: '0' | '1' | '2' | '3' | '4' | '5'
)

convertHeicToJpg (
  heicAsBase64: string,
  fmScript: string,
  scriptOption?: '0' | '1' | '2' | '3' | '4' | '5',
  quality?: number (number between 0 and 1)
)
